Nutrio App: mobile nutrition tracking with visible proof

This section borrows the structured case-file idea from the new version, but uses real local video and PDF evidence instead of simulated verification buttons.

Full Report

Problem Statement

Nutrition tracking becomes inconsistent when users must manually search foods, estimate portions, and log meals across many steps.

Engineered Solution

A Flutter Android MVP that combines scan-first meal logging, barcode-based product lookup, Firebase-backed state, and nutrition API integration.

Key Technical Features

  • Barcode scanning flow for faster food lookup and reduced manual entry.
  • Firebase/Firestore-backed user state and progress tracking.
  • REST API nutrition data integration for food and macro information.
  • Demo evidence showing both scan flow and dashboard/tracking flow.
